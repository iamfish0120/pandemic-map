# 疫情地图

#### 介绍
基于Vue和element-ui搭建的可视化面页 疫情数据调用数据网站上已有的API  echarts可视化展示

Element-ui v2.15.0 https://element.eleme.cn/#/zh-CN

echarts v4.6.0  https://www.echartsjs.com/zh/index.html

数据来源   https://www.tianapi.com/
